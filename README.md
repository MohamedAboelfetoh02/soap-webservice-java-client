# soap-webservice-java-client
Java SOAP web service and client application using WSDL-based communication for arithmetic and numerical sequence processing.

## Overview
This project demonstrates a Java SOAP web service and a client application that consumes the service using WSDL-based integration.

## Features
- SOAP service operations
- Java client consumption
- Arithmetic processing
- Numerical sequence generation
- Servlet-based result display

## Technologies
- Java
- SOAP
- JAX-WS
- WSDL
- Servlets
- GlassFish
- NetBeans

## Learning Outcomes
- Service-oriented architecture
- Web service integration
- Java enterprise web service development
